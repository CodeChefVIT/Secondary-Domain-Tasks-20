## Instructions to run

```
$ npm start

```
## Docker
```
$ docker build -t n0v0cain3/mini-task .
$ docker run -it -p 8000:8000 n0v0cain3/mini-task
```
Pull image in your system
```
$ docker pull n0v0cain3/mini-task:latest
```
